# unity-rtl
Unity-compatible C# library for rendering right-to-left (RTL) text in Unity. Supports Arabic, Farsi, Hebrew, Urdu, and Pashto with contextual shaping, Lam-Alef ligatures, diacritics, mixed LTR/RTL text, and configurable numerals. Includes a Unity UI word-wrap helper. Targets .NET Standard 2.0 and Unity 6.0.
